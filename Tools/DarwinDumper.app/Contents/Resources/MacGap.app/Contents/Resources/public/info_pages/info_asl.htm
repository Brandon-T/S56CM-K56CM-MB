An attempt is made to find an Apple System Log for the last day the system was booted. If found, the last boot log from that day is dumped for a file named BootLog_asl.txt in the latest dump/Kernel Logs directory.

The command used to read the file is syslog -f /var/log/asl/<FILE>

Note The log files are only retained for a set number of days so if the system has been running for longer than this then the file will not be available.
